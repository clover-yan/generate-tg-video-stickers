# generate-tg-video-stickers

This repository helps you convert GIFs to Telegram video stickers using the GitHub Actions workflow.

## Usage

1. Fork this repository.
2. Create a branch with your desired name.
3. Upload your GIF files to the `gifs` directory in the created branch.
4. Go to the "Actions" tab and wait for the workflow to complete.
5. Once the workflow is finished, you can find the converted video stickers in the artifacts section of the workflow run.
6. Download the video stickers and upload them to Telegram as stickers.
